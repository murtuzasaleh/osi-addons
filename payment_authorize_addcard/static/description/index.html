Payment Authorize.net: Add a Card

Module lets the Odoo user manage payment details(credit-card) for a customer.

This module extends the functionality of Odoo v10 to support website payment integration and adds the facility to add payment token for a customer via an Odoo user.
The payment token/credit cards can be created from Contacts, Sales order, and Account Invoice form views.
